## 🎮 Sobre o Projeto

O **JokenPo Game** é um projeto Front-End criado para praticar lógica de programação e manipulação de elementos da interface utilizando JavaScript puro.

O jogador enfrenta a máquina em partidas de Pedra, Papel e Tesoura, onde a escolha da IA é gerada aleatoriamente através do `Math.random()`, podendo resultar em:

- Vitória do jogador
- Vitória da máquina
- Empate

A interface é atualizada dinamicamente conforme o andamento das partidas e da pontuação.
